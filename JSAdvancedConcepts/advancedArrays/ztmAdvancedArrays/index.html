Advanced Arrays

  • const list = [{}, {}, {}] - obj inside arrays
  • These objects can be accessed inside arrays like normal elements (through [0][1] indexing)

forEach

    Executes a provided function ONCE forEACH array element

  • iterates through array, but does not use a "return" value

Map

    Creates a new array with the results of calling a provided function on every element in the calling array

  • iterates through array (same as forEach)
    - does require a "return" value
    - .map returns a new array itself
