Object Constructures

  • Object constructors are used for creating many objects of the same "type" (in this case, a person)

  • The function is an object constructor (used as a blueprint to help create the object), like a template, but does not create the object itself

  • The "This" keyword refers to the object that "owns" the code...
    1. The value of "this" when used in an object, is the object itself
